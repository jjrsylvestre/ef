:: Elementary Foundations

Elementary Foundations is a discrete mathematics textbook with a focus on developing mathematical logic skills. Originally authored in LaTeX for use in AUMAT 250 Discrete Mathematics at the University of Alberta's Augustana Campus, it has recently been converted to PreTeXt for release as an open-access educational resource. Full PreTeXt source code will be made available on github.com in the near future.

Elementary Foundations is free as in “freedom” — released under the GNU Free Documentation License (GFDL), you are free to use, copy, redistribute, and/or modify this textbook. (Though that freedom comes with some responsibilities; see the full text of the GFDL, included as an appendix to the textbook.) If you wish to make use of this work under a different license, please contact the author using the contact info below.
